## My first project involves using Logic and Permutations to solve a puzzle given:    

        
- an array representing the puzzle grid       
        
- specifications required for puzzle grid (ie: what rules does each grid value have to follow, are those linked to another grid index? etc...)              
        
- our puzzle pieces as a list[list[int]] where the list[int] contains two values in range 0-6        
                                                                                                  

                                                                                                         
It should return the solution of the puzzle as a formatted string containing all the game pieces in the right spot of the puzzle grid.       

       
If there is no solution, it should output N/A or raise an ERROR.           




The goal of this project is to learn more about arrays and matrixes, as well as how to limit large time complexities.                            
I'm also hoping to create a more abstract version that can handle nth values for puzzle piece size and range.
